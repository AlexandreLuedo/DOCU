* Files & directories
** Knowing working location
#+begin_src bash
$ pwd                           # Print of Working Directory
#+end_src
** Creating directory
#+begin_src bash
$ mkdir [options] <directory_name> [<other_directory_name>]                         
-p                              # Parents
-v                              # Verbose
-m, --mode=<mode>               # Set file mode (as in chmod), not a=rwx - umask
-Z                              # Set SELinux security context of each created diurectory to the default type
#+end_src
* Services
** Start, restart, enable etc...
#+begin_src bash
$ sudo systemctl [<task>] [<service_name>]
#+end_src
** Read log
#+begin_src bash
$ sudo journalctl -xeu [<service_name>]
#+end_src
