* Extensions
** MCP servers (Model Context Protocol)
Allows the model to interact with externals APIs, databases, or locales applications.
** Agent skills
Directories of expertise. The model "activates" these on-demand when it identifies a matching task, which saves context tokens.
** Hooks
Scripts that run at specific points in the agent's lifecycle (e.g., before executing a tool to validate arguments or after a model response to filter content).
** Custom Commands
User-defined shortcuts (e.g., /my-action) triggered prompt or shell commands
** Context Files
(GEMINI.md): Instructions that define the "personnality" loaded automatically at each startup session.
