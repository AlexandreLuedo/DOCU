* Build & run
Create a docker image => run
#+begin_src bash
# Build the image
$ docker build -t <path/to/project> .
# List the image
$ docker image ls
# Run the image
$ docker container run [OPTIONS] IMAGE [COMMAND] [ARG...]
#+end_src
