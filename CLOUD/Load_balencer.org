#+title: Load Balencer
* Concept
- tools or application
**** Hardware based
Require a dedicated load balancer device
**** Software based
Can run on a server or on a VM. Also in the [[https://www.cloudflare.com/learning/cloud/what-is-the-cloud/][cloud]].
**** Static load balencing algorithm
- Will not be aware of which server is not to used or has to eavy workloads.
- Assign hardware trough a plan
- Quick and easy to setup
**** Dynamic load balencing algorithm
- More difficulte to setup
- Take care of the workload, availability, health
