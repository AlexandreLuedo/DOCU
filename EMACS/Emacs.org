** Search & Replace (Vim / Evil mode)
=:%s/<string>/<string>/g=
- =:= emacs command mode
- =%= apply on all the file
- =s= mean substitue
- =g= mean global (modify all lines)
- =c= ask for confirmation before deleting
*Note:* In regexp, if you use {{ }} in replacement, add =\= before the caracter: =:%s/<\{\{ string \}\}>/<string>/g=
** Remove last caracters of each lines

*** Display buffered time
#+begin_src elisp
(defun open-clock-buffer ()
  "Open a buffer displaying the current time."
  (interactive)
  (let ((buf (get-buffer-create "*Clock*")))
    (with-current-buffer buf
      (erase-buffer)
      (insert (format-time-string "%A, %d %B %Y\n%H:%M:%S"))
      (goto-char (point-min))
      (display-buffer buf)
      (run-with-timer 1 nil (lambda ()
                              (with-current-buffer "*Clock*"
                                (erase-buffer)
                                (insert (format-time-string "%A, %d %B %Y\n%H:%M:%S"))
                                (goto-char (point-min))))))))
#+end_src
