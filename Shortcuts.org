#+title: Shorctus
* VIM

** Exit the editor
- ZZ :: exit & save
- wq :: exit & save
- x :: exit & save
- w <name> :: save in the <name> file (without exiting the ide)
- 1,10w <name> :: save line 1 to 10 in the <name> file
** Structuring vim sentences
- *b* + *number* + *motion
- b = an operator
- Motion = like $ w e

**  Moving
*** This is the motions
- $ = end of the line
- w = forward word
- e = end of the current word
- 0 = move to start of the line
- G = move to the bottom
- 12G = got to the 12 line
- gg = start of the page

**  Deleting
- x = delete letter
- d = delete
- dd = delete line
- D = delete after cursor

** Inserting
- i = insert
- a = insert but after one letter
- o = insert below the cursor
- O = insert upper the cursor
- p = insert one or multiples lines below
- P = insert one or multiples lines above
** Undo & Redo
- u = undo
- U = fix the whole line
