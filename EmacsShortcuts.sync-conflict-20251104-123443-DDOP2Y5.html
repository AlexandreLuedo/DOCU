VIM

Structuring vim sentences

  • b + number + motion
  • b = an operator
  • motion = like $ w e

Moving

This is the motions

  • $ = end of the line
  • w = forward word
  • e = end of the current word
  • 0 = move to start of the line
  • G = move to the bottom
  • 12G = go to line 12
  • gg = start of the page

Deleting

  • x = delete letter
  • d = delete
  • dd = delete line
  • dw = delete word

Inserting

  • i = insert
  • a = insert but after one letter
  • o = insert below the cursor
  • O = insert upper the cursor

Undo Redo

  • u = undo
  • U = fix the whole line
  • C-r = redo

Copy, paste & replace

  • y = copy/yank
  • yy = yank line
  • p = for pasting the delted content
  • rx = for remplacing the cursor selected letter with the “x”
  • ce = change until the end of the line
  • cc = the whole line

Search

  • / = search trhough the doc
  • ? = search to the backward position
  • n = search the last thing you search
  • N = same but in the opposite position
  • C-0 = to go back where you are
  • % = search the end of an { ( [ ] ) }

Modify file type

  • s = substitute :s/old/new/g to substitute ’new’ for ’old’

Commands

  • :! = type an terminal/external command
  • r = retrieving an file (usefull when you make shit)
  • R = to remplace than more than one caracter

Tips

v + :w nameOfTheFile = you can select an precize part and save it.
