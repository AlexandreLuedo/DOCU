* Global Navigation
- ~1~ to ~5~ :: Jump directly to panels (1: Status, 2: Files, 3: Branches, 4: Commits, 5: Stash).
- ~[~ / ~]~ :: Cycle through tabs within a panel (e.g., Local/Remotes).
- ~h~ / ~l~ :: Navigate the tree view (Collapse/Expand folders).
- ~Enter~ :: Zoom into the active panel or enter a directory.
- ~Esc~ / ~q~ :: Go back or quit the application.
* File Management (Status)
The main area for staging and preparing commits.

- ~Space~ :: *Stage / Unstage* (Toggle file selection for the index).
- ~a~ :: *Stage All* (Add all modified files).
- ~d~ :: *Discard* (Revert all changes for the selected file).
- ~c~ :: *Commit* (Open the simple commit dialog box).
- ~C~ :: *Commit* (Open the full editor for the commit message).
- ~i~ :: *Ignore* (Add the file to .gitignore).

* Branches & Sync
- ~Space~ :: *Checkout* (Switch to the selected branch).
- ~n~ :: *New Branch* (Create a new branch from the current one).
- ~p~ :: *Pull* (Fetch and merge remote changes).
- ~P~ :: *Push* (Send local commits to remote).
- ~M~ :: *Merge* (Open the merge menu for the selected branch).

* Advanced Actions
** Stash & Reset
- ~s~ :: *Stash* (Save changes temporarily).
- ~g~ :: *Reset* (Open the reset menu: Soft, Mixed, Hard).

** History & Corrections
- ~f~ :: *Fixup* (Mark as a fixup for the previous commit).
- ~R~ :: *Rename* (Rename a commit or a branch).

* The Secret Weapon
- ~x~ :: **Context Menu** (Opens a list of all possible actions for the item under the cursor).
- ~?~ :: **Global Help** (Displays all keyboard shortcuts).
