* Issue
** Create
#+begin_src bash
gh issue create
gh issue create --title "<issue_title>" [--body "<body>"]
# template
gh issue create --template "bug_report.md"
#+end_src
** View
#+begin_src bash
gh issue view <issue_tag_only_integer> [<--web>]
#+end_src
** List
#+begin_src bash
gh issue list
#+end_src
** Close
#+begin_src bash
$ gh issue close 123 -c <string> -r <string>
# -r flag include 
#+end_src
** Subissue
#+begin_src bash
$ gh sub-issue create --parent 123 --title "title" --body "body"

  --title "Titre de la PR" \
  --body "Description détaillée de la PR.

  - Référence ticket #123
  - Points clés…
  "
