** Setup git
#+begin_src bash
$ git init # Cela va créer un répertoire
#+end_src
** Ajout d'un fichier dans le répertoir git
#+begin_src bash
$ git add mon.fichier # Ajoute le fichier au répertoire
$ git add * # tout les fichiers
#+end_src
** Suppression d'un fichier dans le répertoir git
#+begin_src bash
$ git rm --cached chemin/vers/fichier.ext
#+end_src
** Voir le status
#+begin_src bash
$ git status
#+end_src
** Valider un changement / commmit
#+begin_src bash
$ git commit -m "commentaire" # le flag -m défini un commentaire
#+end_src
** Paramètres
#+begin_src bash
$ git config color.ui true # Ajoute des couleurs
#+end_src

* Setup un repo depuis github
** http
#+begin_src bash
git init
git remote add origin https://github.com/ton-nom-utilisateur/nom-du-repo.git

git commit -m "Premier commit du projet Flutter"
git branch -M master
git push -u origin master

* Gestion de branch
** Suppr une branch en local
#+begin_src bash
git branche -d main
#+end_src
** Suppr en distant (github)
#+begin_src bash
git push origin --delete main
#+end_src
** Pousser origin sur une branch
#+begin_src bash
git push -u origin master
#+end_src
** Merge une branche
#+begin_src bash
$ git checkout master #or# $ git switch master # s'assurer d'être sur la bonne branche
$ git pull origin main #or# $ git pull origin master # Mettre à jour le git
$ git merge [source_branch] # quand l'on est sur la bonne branche
# Options:
# -m pour attribuer un message
$ git push origin main #or# $ git push origin master # pour mettre à jour pour les autres membres de l'équipe.
#+end_src

** Vérifier si la branche est fusionnée
#+begin_src bash
$ git branch --merged master
$ git branch --merged
$ git branch --no-merged
# Par défaut, cela va s'appliquer aux branches locales uniquement.
# Options:
# -a va montrer les branches locales & distantes
# -r va montrer les branches distantes
#+end_src
